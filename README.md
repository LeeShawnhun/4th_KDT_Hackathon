# 2023년 제4회 K-디지털 트레이닝 해커톤
## 주제: 기상청 API를 활용한 닻끌림(주묘현상) 예측
- 맡은 역할: 데이터 전처리, 딥러닝 모델 구성
## 컨텐츠 설명
- **Hackathon**: Spring Boot 4 기반의 web service 구현 파일
- **data**: 딥러닝 모델 구현에 사용된 데이터들
- **others**: 모델 학습에 필요한 데이터 전처리 및 모델 구현 과정 코드
